## ⚛️ 量子通信核心技术

### 1. 量子比特 (Qubit)
```javascript
class Qubit {
  constructor(alpha = 1, beta = 0) {
    // 量子态: |ψ⟩ = α|0⟩ + β|1⟩
    this.alpha = alpha;  // |0⟩ 的振幅
    this.beta = beta;    // |1⟩ 的振幅
  }

  measure() {
    // 波函数坍缩
    // 概率: P(0) = |α|², P(1) = |β|²
  }
}
```

**特性**:
- 叠加态: 同时处于 |0⟩ 和 |1⟩
- 测量坍缩: 观测导致波函数坍缩
- 概率性: 测量结果是概率的

### 2. 量子纠缠对
```javascript
class EntangledPair {
  constructor() {
    // Bell 态: |Φ⁺⟩ = (|00⟩ + |11⟩)/√2
    this.qubitA = new Qubit(1/√2, 1/√2);
    this.qubitB = new Qubit(1/√2, 1/√2);
  }

  measure() {
    // 瞬时关联: 无论相距多远
    // Alice 测量 |0⟩ → Bob 瞬间得到 |0⟩
  }
}
```

**特性**:
- 非局域性: 超越空间限制
- 瞬时关联: 测量一方，另一方立即确定
- 不可克隆: 无法复制量子态
